Server Requirements
======

Web
------
* PHP version 7.2 or newer.
* GD PHP
* OpenSSL PHP
* CURL PHP
* Intl PHP
* GMP PHP


Dedicated Server (Linux)
------

* Installed GameAP Daemon


Dedicated Server (Windows)
------

* Installed GameAP Daemon
* Administration privileges
